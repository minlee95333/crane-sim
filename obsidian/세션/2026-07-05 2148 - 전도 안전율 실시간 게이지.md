# 전도 안전율 실시간 게이지

## 요청

- P7.12+ 백로그 첫 항목으로 매달린 하중의 전도 안전율 순수 질의와 실시간 색상 게이지를 구현한다.
- 안정성 계산은 기존 `Stability.js`를 재사용하고 렌더 계층에서 재계산하지 않는다.

## 수행

- `World.stabilityPreview(craneId)`를 추가해 매달린 하중과 현재 반경·붐 길이로 `checkStability`를 호출하고 `tippingMargin`을 반환했다.
- `World.getState()`의 크레인 스냅샷에 `stabilityFactor`를 실어 렌더 계층의 읽기 전용 계약으로 연결했다.
- `ScreenWidgets` 게이지에 안전율 값과 색상 바를 추가했다. 1.33 이상 녹색, 1.0 이상 1.33 미만 호박색, 1.0 미만 적색이다.
- 순수성, 기존 계산 재사용 결과 일치, 상태 스냅샷 전달, 세 임계 구간 회귀 테스트를 추가했다.
- `SIM_DESIGN.md` P7.12+를 Tier1 완료·전체 진행 중으로 갱신했다.

## 변경 파일

- `src/core/World.js`
- `src/core/stabilitypreviewtest.js`
- `src/render/ScreenWidgets.js`
- `src/render/viewtest.js`
- `SIM_DESIGN.md`
- `obsidian/프로젝트 개요.md`

## 검증

- `npm test`: 23개 테스트 파일, 462개 단정 통과.
- `npm run build`: 성공, 47개 모듈 변환.
- 경고는 기존 500kB 초과 번들 경고 1건뿐이며 새 경고는 없다.

## 결정과 근거

- 별도 모멘트 식을 만들지 않고 `checkStability`의 `tippingMargin`을 그대로 사용해 계산 진실 원천을 하나로 유지했다.
- 게이지는 `getState()`가 제공한 `stabilityFactor`만 사용하므로 core → render 계층 경계를 지킨다.
- 하중이 없거나 안정성 질량 제원이 없으면 `null`을 반환해 의미 없는 게이지를 숨긴다.

## 남은 작업

- P7.12+ Tier2·Tier3 백로그는 미착수다.
- 브라우저 수동 시각 검증은 이번 완료 조건에 포함되지 않아 수행하지 않았다.
